# Booking API

This is a backend API for managing hotels, rooms, and bookings.

## Setup

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/booking-clone-api.git
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Create a `.env` file based on `.env.example`:
    ```bash
    cp .env.example .env
    ```

4. Start the server:
    ```bash
    npm start
    ```

## Documentation

You can view the API documentation at:
[http://localhost:3000/docs](http://localhost:3000/docs)
